# memory/

- **`MEMORY.md`** — curated working memory (gitagent layer; keep under ~200 lines). See `memory.yaml`.
- **Dated files** `YYYY-MM-DD.md` — narrative logs; routine sessions load **today + yesterday** only.
- **`runtime/`** — short cross-session state (`dailylog.md`, `context.md`, `key-decisions.md`).
- **`archive/`** — optional rotated snapshots.
