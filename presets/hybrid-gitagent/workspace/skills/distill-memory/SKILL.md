# Distill memory

When the human says a correction is permanent:

1. Add to today’s `memory/YYYY-MM-DD.md`
2. If it affects all agents, add to `shared-context/FEEDBACK-LOG.md`
3. After repetition, add a bullet to root `MEMORY.md` or the relevant agent `MEMORY.md`
4. Trim obsolete bullets during review
