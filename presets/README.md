# Presets

| Folder | Use when |
| ------ | -------- |
| [`light/workspace`](light/workspace) | One agent, fastest start |
| [`full/workspace`](full/workspace) | Multi-agent intel → drafts, no gitagent extras |
| [`hybrid-gitagent/workspace`](hybrid-gitagent/workspace) | Same team as **full** + `agent.yaml`, `RULES.md`, `DUTIES.md`, `skills/`, `workflows/`, hooks, structured memory |

Copy via repo root `scripts/copy-preset.ps1` / `copy-preset.sh`, or duplicate a folder manually.
