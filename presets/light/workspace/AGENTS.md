# AGENTS.md (root)

## Every session

Before doing anything else:

1. Read root `SOUL.md` (if present) or your assigned agent `agents/<slug>/SOUL.md`
2. Read `USER.md`
3. Read `memory/YYYY-MM-DD.md` for **today** and **yesterday** (create today if missing)
4. In **main / direct** sessions only: also read `MEMORY.md` if it exists

## Memory

- Mental notes do not survive session restarts; files do.
- When the human says "remember this", append to today's daily log and, if it is durable, add a line to `MEMORY.md` after distill.
- Prefer text in files over unstated assumptions.

## Safety

- Do not exfiltrate private data.
- Prefer recoverable deletes over destructive commands when unsure.
- When in doubt, ask the human.
